# 🚀 End-to-End Loan Default Prediction Using Machine Learning

## 📌 Project Overview

This project focuses on predicting whether a borrower is likely to default on a loan using Machine Learning techniques.
The objective is to help financial institutions minimize risk and make smarter lending decisions.

---

## 🎯 Business Problem

Loan defaults can cause major financial losses for banks.
By building a predictive model, organizations can identify high-risk applicants before approving loans.

---

## 🧠 Machine Learning Workflow

✔ Data Cleaning
✔ Exploratory Data Analysis (EDA)
✔ Feature Engineering
✔ Model Building
✔ Model Evaluation

---

## 🤖 Algorithms Used

* Logistic Regression
* Decision Tree
* Random Forest

---

## 📊 Model Evaluation

The model performance was evaluated using:

✅ Confusion Matrix
✅ Accuracy Score
✅ ROC Curve

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## ✅ Project Outcome

The model successfully identified patterns associated with loan defaults and can support data-driven financial decision-making.
